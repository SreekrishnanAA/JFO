What is JFO

JFO helps to automate the process of scanning a specified directory,catoegorizing files into distinct folders based on their extension

and establish a well-organized directory for improved file management.

The organizer is designed to automatically sort and categorize files in a user- specified directory, making it easier to find and access specific files.

The program utilizes various Python modules, including the os and path modules, to navigate the file system and move files to designated folders.

The user is able to customize the organization scheme and specify the types of files to be sorted. The program also includes a feature for removing duplicate files.

The organizer has been tested and found to be effective in improving the organization and efficiency of the user's file system.

The program begins by allowing the user to select a directory to organize.

Once the directory is selected, the program scans all the files within it and categorizes them based on their file type, such as documents, images, videos, etc.

The program also allows the user to specify additional custom categories if desired.

The program then moves the files into their respective category folders, creating the folders if they do not already exist.

The program also has a feature for managing and cleaning up empty folders.

If the program detects any empty folders within the selected directory, it prompts the user to delete them.

This helps to keep the file system clean and free of unnecessary clutter.

Overall, the program makes it easy for the user to manage and organize their files, and it is a useful tool for anyone who wants to keep their file system in order.

It is easy to use and can be customized to fit the user's specific needs
